### E-commerce via Java Spring Boot Gradle and React Vite
* **
##### Overview:

* A web application was created with a modern design based on Java Spring Boot Gradle and React, covering multiple framework technologies. During this process, a modern design was achieved based on the wide range of Spring Boot technologies and, of course, on the front-end, also based on TailwindCSS.
* **
##### Category:
1. Admin Frontend
2. Client Frontend
3. Server
* **
##### Tech Stack:
###### Frontend:
* Vite
* Tailwindcss
* Docker
* Nginx
###### Server:
* Spring Boot
* Gradle
* PostgreSQL
* Redis
* RabbitMQ
* SMTP
* JWT(Json Web Token)
* **
##### Business Logic for Server:
1. Authentication System
2. Product Management
3. Cart Management
###### 1. Authentication System:
* Register
* Login
* Refresh
* Logout
* Refresh
* Forgot Password
* Reset Password
* Verify Email
###### 2. Product Management:
* Create product
* Get a product
* Get product list
* Delete a product
* Update a product
###### 3. Cart Management
* Create cart
* Add product to cart
* Remove product from cart
